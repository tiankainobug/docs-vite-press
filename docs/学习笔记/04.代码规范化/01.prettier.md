# Prettier

## 一. 安装步骤

### 1.安装插件

WebStorm 默认装了 **prettier 插件**。不需要我们自己装了。

### 2.安装node包

这里我使用pnpm进行安装

命令：
```node
pnpm add preffier -w -DE
```
解释：
- -w： 使用了 monorepo 框架，-w 安装在根工作空间
- -DE：D为开发环境，E 为锁定版本号
## 二.配置

### 1. 编写配置文件

主要有两个配置文件：
- `.prettierrc.js`
- `.prettierignore`

其中 `.prettierrc.js`文件内容如下:
```js
const config = {  
    // 每行最大长度  
    printWidth: 100,  
  
    // 缩进空格数  
    tabWidth: 4,  
  
    // 使用空格而不是制表符  
    useTabs: false,  
  
    // 语句末尾是否加分号  
    semi: false,  
  
    // 是否使用单引号  
    singleQuote: true,  
  
    // 对象属性是否加引号: “as-needed” 仅在必须时添加  
    quoteProps: 'as-needed',  
  
    // 多行尾部是否加逗号：ES5  
    trailingComma: 'es5',  
  
    // 对象左右空格 { foo: bar }    bracketSpacing: true,  
  
    // 箭头函数参数是否总是加括号  
    arrowParens: 'always',  
  
    // 换行风格（auto、lf、crlf）  
    endOfLine: 'lf',  
  
    // Vue 文件中 <script> 和 <style> 的缩进  
    vueIndentScriptAndStyle: false,  
  
    // 是否格式化嵌入的代码，比如 markdown 中的代码块  
    embeddedLanguageFormatting: 'auto',  
}  
  
export default config
```


`.prettierignore`文件如下：
```.ignore
node_modules/  
dist/  
build/  
.env
```
