# 设计思想和理念
## vue3的特点

1.  声明式编程框架
2. 虚拟 DOM
3. 区分运行时和编译时
	- 运行时 Runtime：
		- 渲染函数 -> 虚拟 DOM -> 真实 DOM
	- 编译时 compile time:
		- template -> 真实 DOM
	- Vue3 对编译时和运行时的清晰拆分，是其性能优于 Vue2 的重要原因之一 —— 通过编译时的优化，让运行时的工作更高效。

## 设计思想

1. 拆分模块
2. 组合式 api
3. 自定义渲染器
4. 采用 RFC [[02.RFC 是什么 | RFC 介绍]]
