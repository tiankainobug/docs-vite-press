## Reflect使用

```js
import {reactive} from "@vue/reactivity";  
  
const person = reactive({  
    name: '张三',  
    get aliasName() {  
        return this.name + '帅气'  
    }  
})  
  
const proxyPerson = new Proxy(person, {  
    get(target, key, receiver) {  
        console.log('key:', key)  
        // target[key] 不会触发get  
        // receiver[key] 会一直触发get 死循环  
        return Reflect.get(target, key, receiver)  
    },  
})  
  
console.log(proxyPerson.aliasName)
// aliasName
// name
// 张三帅气
```

>Reflect.get(target, key, receiver)   // 更改 this 的指向为 receiver 而不是target

使用Reflect 来更改 getter 和 setter 中 this 的指向避免死循环
